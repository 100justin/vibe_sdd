# 데이터 모델: 오늘의 할 일 웹 앱

## 엔티티

### Todo
할 일 항목을 나타내는 핵심 엔티티다.

**속성**:
- id: 식별자
- title: 할 일 제목
- completed: 완료 여부
- created_at: 생성 시각
- updated_at: 수정 시각

## 관계

- 하나의 할 일 항목은 독립적으로 존재한다.
- 할 일 항목은 완료 여부와 필터 상태에 따라 화면에 표시된다.

## 제약 조건

- title은 비어 있지 않아야 한다.
- completed는 기본값 false로 저장된다.
- 삭제된 항목은 목록에서 제거된다.
